# Contributing to Awesome Agentic Income

Thanks for helping build the best list of honest AI-income resources.

## Guidelines

- **Honesty first.** Only suggest tools, programs, or guides you'd genuinely recommend. This list's value is that it doesn't shill.
- **One addition per pull request.** Easier to review, easier to discuss.
- **Keep links live.** No dead links, no redirects through trackers in the list itself.
- **Match the format.** Use the existing table/bullet style of the section you're adding to.
- **Say why it belongs.** One short clause on what makes the entry worth a reader's time.
- **Affiliate programs:** include commission + recurring terms, and link the official signup page. Note that terms change — accuracy matters more than optimism.

## What doesn't get merged

- Tools with no real use case beyond a referral payout.
- Closed or dead affiliate programs (mark existing ones if they close).
- Anything that asks readers to act against their own interest.

Open a PR. Be honest. That's the whole bar.
